## 🚀 Overview

EduAssist Chatbot is a rule-based educational assistant designed to answer basic student queries related to technology and academics.

The chatbot uses predefined response mappings and conversational logic to simulate interactive question-answer sessions with users.

---

## 🧠 Technologies Used

* Dictionaries
* Random Module
* Conditional Logic
* Command-Line Interface (CLI)

---

## 📊 Features

* Student query assistance
* Interactive chatbot conversations
* Randomized responses
* Technology-related knowledge base
* Continuous conversation loop
* User-friendly command-line interface

---

## ⚡ How It Works

User enters a message

↓

Chatbot checks predefined response database

↓

Matching response category is identified

↓

A response is selected randomly

↓

Bot replies to the user

↓

Conversation continues until exit command
